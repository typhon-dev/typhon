# Typhon Language Decision Records

## Create a new TLDR

For new TLDRs, the status should be `Proposed`. Use one of the following methods to automatically generate a markdown document for the TLDR.

### VSCode Snippet

1. Create a new markdown file in this directory. The file should follow the naming format `NNNN-lowercase-title-with-dashes.md`
1. Open the file in a VSCode editor
1. Type `tldr` and select it from the editor suggestions (may need to show suggestions with `CTRL + space` or `command + space`)
1. Use tab to navigate through the fields requiring input and update the placeholder text

## Submit TLDR

Create a pull request that adds the new TLDR file to the repository.
